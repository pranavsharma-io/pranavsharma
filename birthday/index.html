🎂 About Birthday Data Master

Why This Exists

The DAV Jehanabad website shows daily birthdays, but the list disappears after the day. This app archives it permanently.

How It Works — Backend

  1. GitHub Actions triggers a Python script every day at 11:00 AM IST.
  2. The script scrapes davjehanabad.in and downloads birthday HTML.
  3. A second script parses names, classes, parents, dates into a clean .xlsx file.

How It Works — Frontend

  1. Browser fetches the .xlsx from birthday-api.pages.dev.
  2. XLSX.js reads it client-side — no server required.
  3. All filtering, sorting, and search run locally — instant response.

Features

✔ Month + class + section filters with sidebar drill-down
✔ Sibling finder via fuzzy parent name matching
✔ Today's birthdays highlighted + dedicated tab
✔ Automatic duplicate suppression
✔ Sortable columns, pagination, stats dashboard
