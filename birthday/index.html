About Birthday Data Master

Why This Project Exists

The official DAV Jehanabad website shows daily birthdays, but the list disappears after the day ends. This app turns that fleeting list into a permanent, searchable archive.

How It Works

Backend (Automated Engine)

  1. Automated Trigger: GitHub Actions runs a Python script every day at 11 AM IST.
  2. Scraping: The script downloads raw HTML from davjehanabad.in.
  3. Formatting: A second script extracts names, classes, parents, and dates into a clean Excel file.

Frontend (Your Browser)

  1. Fetch: Your browser downloads the Excel file once.
  2. In-Browser Processing: XLSX.js reads the file and builds the table — no server needed.
  3. Instant Search & Filter: All filtering happens locally — zero delay.

New Features (Enhanced Version)

✦ Year-aware search — filter by month and class simultaneously.
✦ Sibling Finder — fuzzy-match parent names to find all children in one family.
✦ Today's Birthdays — auto-highlighted banner and dedicated tab.
✦ Duplicate suppression — identical entries are hidden automatically.
✦ Stats dashboard — quick overview of distribution by class and month.
